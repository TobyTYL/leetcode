# 滑动窗口

* 目的：滑动窗口算法技巧主要用来解决子数组问题，比如让你寻找符合某个条件的 __最长/最短子数组__
* 思路：维护一个窗口，不断调节窗口的大小和位置，更新答案
* 模板：
    ```python
    left, right = 0, 0
    while right < len(s):
        # 增大窗口
        window.add(s[right])
        right += 1

        while window needs shrink:
            # 缩小窗口
            window.remove(s[left])
            left += 1
    ```

* O(n)解法：滑动窗口的时间复杂度是O(n)，因为每个元素只会进入窗口一次，而且左右指针每次只会移动一次，left和right指针并没有回退过
